Arduino_FMRadio
===============

Simple FM radio made on Arduino + RRD-102 V2 (RDA5807M)
Components:
FM Stereo Module Radio Module RDA5807M RRD-102 V2.0
Arduino Nano
Display Hitachi HD44780
Computer speakers or headphones
